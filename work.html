Web App: Pokedex
I developed this web app with HTML, JavaScript, and CSS, and I have integrated the jQuery library and Bootstrap framework into this app. It fetches from a Pokemon API and displays the Pokemon as buttons. The buttons can be clicked to trigger modal functions built into Bootstrap. The modals display the Pokemon images, names, types, and heights.

Progressive Web App: Meet
For this project, a big focus was on test-driven development and behavior-driven development. This project represents my first big dive into writing serverless functions, unit tests, integration tests, end-to-end testing, and user acceptance tests. For authenticating, I made use of Google OAuth authentication. I was able to implement Object-Oriented Programming with my alerts. Being that this is a Progressive Web Application, it is able to run offline and can be downloaded to your computer or mobile device. Last, I was able to integrate data visualizations with Recharts.
